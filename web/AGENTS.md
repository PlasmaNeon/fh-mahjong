# web/

> React frontend application for the Mahjong tabletop UI.

## Overview

A single-page React application built with Vite and TypeScript. Renders the mahjong table, player hands, tile animations, action buttons, and scoring modals. Communicates with the Go backend via WebSocket using binary Protobuf messages. Styled with TailwindCSS, animated with Framer Motion.

## Key Files

- **package.json** — Dependencies: React 19, Vite 7, TailwindCSS 4, Framer Motion 12, protobufjs 8
- **vite.config.ts** — Vite bundler configuration
- **tsconfig.json** — TypeScript configuration
- **index.html** — HTML entry point

## Subdirectories

- **src/** — Application source code (pages, contexts, hooks, utils, proto bindings)
- **public/** — Static assets (tile SVGs in `Regular_shortnames/`, WASM binary, `wasm_exec.js`)
- **dist/** — Build output (generated by `npm run build`)

## Architecture Notes

- Dev server: `npm run dev` (Vite on port 3000, proxies API to `:8080`)
- Build: `npm run build` (outputs to `dist/`)
- All game state arrives as Protobuf binary via WebSocket; decoded by `protobufjs`.
- Tile rendering uses layered SVG images: `Front.svg` (background) + tile face SVG.
